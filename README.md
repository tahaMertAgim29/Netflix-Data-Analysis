# Netflix-Data-Analysis

## Introduction

This is a basic data analysis project which uses a Netflix dataset taken from Kaggle

## Features

- Analyzing Dataset
- Data cleaning
- NaN value Analyzing
- Data Visualization
- Queries using Pandas

## Prerequisities

- Visual Studio Code (for IDE)
- Jupyter Notebook
- Python Pack for Visual Studio Code
- Python
- Pandas
- Matplotlib
- Seaborn


## Dataset

`Dataset Link:` https://www.kaggle.com/datasets/swatikhedekar/exploratory-data-analysis-on-netflix-data
